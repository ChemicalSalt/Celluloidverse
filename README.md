# Celluloidverse







Celluloidverse is a web project that allows users to send messages through a form and for admins to view those messages securely. It uses Firebase for backend services like database and authentication.





## Features





- User message submission form


- Admin login using Firebase Authentication


- Admin dashboard to view all user messages stored in Firebase Firestore


- Data stored securely on Firebase


- `.gitignore` configured to hide sensitive files like environment variables





## How to Use





1. Clone this repository to your local machine


2. Open the project folder in your code editor (like VS Code)


3. Open `public/index.html` in your browser to use the site





*(You do not need to install anything if you just want to test the front-end locally)*





## Live Site





This is the full live project hosted on Firebase or any hosting service you use.
